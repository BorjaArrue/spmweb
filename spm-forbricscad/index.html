Key Features

Import spatial data into BricsCAD

Imports spatial data, into new or existing BricsCAD drawings, as BricsCAD entities and Extended Entity Data (EED).

Export spatial data from BricsCAD

Export entities from the drawing to spatial files or databases, and saves their Extended Entity Data (EED) as alphanumeric data tables.

Data management, query and edition

Manage the alphanumeric data linked to the entities, design and edit the structure which will be used to store the data.

Coordinates transformation

Calculate geometric transformations of the features in line with the import and export processes.

Background Maps

Choose from among many Backgrounds Maps from many providers such as OpenStreetMap, Bing, MapBox, Ordnance Survey, etc.

Publishing to Google Earth

Export all or part of the entities in the visible Layers of the drawing and their data, to a KML or KMZ file.

Import tasks

Save Tasks to replay the import processes so many times as desired.

Multi language

Translated into English, German, Spanish and Portuguese.
